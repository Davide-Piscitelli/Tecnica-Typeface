# Tecnica Typeface Specimen - Architecture & Design System

This document outlines the technical specification, semantic markup rules, and style sheet mechanics governing the interactive Tecnica Typeface Specimen web interface ('index.html', 'style.css', and companion script).

---

## 1. Layout Framework & Typography

* The 8px Grid Rule:
  To maintain a rigorous industrial aesthetic reminiscent of legacy system terminals, all layout elements—including paddings, margins, grid gaps, and sizing—are strictly calculated as multiples or sub-multiples of 8px (e.g., 8px, 16px, 24px, 32px, 48px). This ensures absolute mathematical rhythm across the viewport.

* Typographic Stack:
  - System UI: 'Fragment Mono' (with generic fallback system monospace fonts) is implemented to render precise data parameters, metadata labels, and structural controls, enforcing a clean command-line readout look.
  - Specimen Engine: The 'Tecnica' font family core assets, including 'Tecnica-53', 'Tecnica-55', 'Tecnica-53Line', and 'Tecnica-55Line'.

* Hierarchy Scale:
  - 12px (9pt): Secondary metadata, micro labels, cell captions.
  - 16px (12pt): Primary informational paragraphs and body readouts.
  - 24px (18pt): Secondary headings and section chapter indicators.
  - 48px (36pt): Main system title for primary identification.

* Color Design System (Dual-Theme Architecture):
  The interface relies on reactive design tokens to switch seamlessly between two contrast modes:
  
  Light Theme (Default)                 Dark Theme
  ---------------------------------     ---------------------------------
  bg-primary:     #F0F4F8               bg-primary:     #12161A
  text-primary:   #12161A               text-primary:   #F0F4F8
  text-secondary: #475569               text-secondary: #94A3B8
  accent:         #0284C7               accent:         #38BDF8

## 2. DOM Architecture ('index.html')

The markup processes layout elements sequentially using clean semantic HTML tags to eliminate overlapping flows:

* Document Header (<head>): Calibrates mobile viewports and establishes a fast, asynchronous pre-connect connection to the Google Fonts API to retrieve UI monospace assets.
* System Registration Table: A structural, industrial-style metadata block with absolute borders detailing the specimen ID, unique doc code (TT-LINE-5355), designer credits, and the active theme switch cell.
* Section 01 // Live Specimen Field:
  - Control Console (.control-panel): A three-column CSS Grid containing interactive inputs (font weight dropdowns, range sliders scaling size from 12pt to 144pt, and line-height adjusters starting at 1.2).
  - Editable Canvas (#preview-box): An isolated, user-focused container set with 'contenteditable="true"' to allow real-time custom string testing.
* Section 02 // Type Scale View (#cascadeSpread): A dynamic DOM node controlled via JavaScript that generates a sequential typographic ladder from 12pt up to 120pt, with text nodes prefixed as "SIZE_NNPT".
* Section 03 // Character Set Inspection (#inventorySpread): An automated grid matrix dividing uppercase, lowercase, numbers, punctuation, and extended symbol glyphs into separate, easily readable lookup cells.
* Section 04 // Information & Technical Specifications (.columns-2): A dual-column layout mapping out the history, grid constraints, and digital applications of the typeface.
* Section 05 // Supported Languages: A clean, justified text layout documenting the global multilingual formatting support of the glyph set.
* Section 06 // Copyright & Disclosure Footer: Displays licensing terms under the SIL Open Font License 1.1 alongside an action button linking back to the official GitHub source repository.

## 3. Style Sheet Mechanics ('style.css')

* Global Reset Layout Engine: Enforces 'box-sizing: border-box' globally to preserve strict grid borders without expanding elements unexpectedly. The top-level body handles smooth CSS transitions during background and color palette shifts.
* Asset Optimization Pipeline (@font-face): Loads the local font assets ('Tecnica-53', 'Tecnica-55', 'Tecnica-53Line', and 'Tecnica-55Line') using 'font-display: swap' to ensure immediate rendering stability without blocking layout display during initialization.
* Theme Scoping Engine: Leverages data attribute hooks (`html[data-theme='light']` and `html[data-theme='dark']`) to override style variables instantly via JavaScript events.
* Interface Layout Integrity: The structural configuration for `.theme-zone-cell` and `.sys-btn-theme` ensures the theme switch button scales to 100% width fluidly. Borders inherit natively from the parent table's 'border-collapse' rule to prevent doubled border lines, keeping visual grids razor-thin (<= 1px).
* Typography Constraints (.cascade-sample-node): Uses 'white-space: nowrap', 'overflow: hidden', and 'text-overflow: ellipsis' within cascade tables to prevent exceptionally long text entries from overflowing layout boxes.
* Fluid Layout Breakpoints (@media): At a max-width threshold of 768px, media queries break strict table components into responsive 100%-width block rows, restructuring control panels vertically and modifying screen padding to prioritize mobile and touch interaction.
